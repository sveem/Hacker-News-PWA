# Hacker News PWA
## Description: 
Hacker News Progressive Web App
## Tech Stack
* Angular 5
* Angular Material 
* Service Workers
### How to run:
Install npm dependencies:
```
npm install
```
Run the app 
```
ng build --prod
cd dist
http-server -p 8080

Workflow  
```
Open Chrome and visit localhost:8080
Refresh all news 
Go Offline
Add new jobs, create & select news
Close the browser
Go Online and open Chrome 
Visit localhost: 8080
